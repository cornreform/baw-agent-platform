# Contributing to BAW

Thanks for stopping by. BAW is a personal project built for our own use, but if something doesn't work or could be better, we'd love to hear from you.

## How to contribute

- **Issues and ideas** are always welcome. Open a GitHub issue for bug reports, feature suggestions, or questions — we'll take a look.
- **Pull requests** are welcome too, but please open an issue first to discuss what you'd like to change. This way we can make sure it fits before you invest time in the code.
- **No CLA, no complex process.** Keep it simple. If you see something broken or missing, just let us know.

## Guidelines

- Be respectful. This is a small project maintained by a handful of people.
- For bugs, include enough detail to reproduce (OS, Python version, BAW version, steps, logs).
- For features, a short description of what you'd like and why is enough.

That's it. Thanks again.
