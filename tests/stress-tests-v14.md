# BAW v0.14.0 壓力測試

## 測試原則
- 記錄每次呼叫的時間、token 使用、是否 crash
- 可接受少許延遲，但不可 crash 或無限 loop

---

```
你好
```

```

```

```
1. 你好
```

【預期】
- 所有回覆正确
- 最後一條應該搜到野餐記憶

## S2 — 極端長 Multi-Task

### S2a — 15+ 項任務
```
今日有以下工作要做，幫我完成：
1. 記低：我鍾意食拉麵
2. 讀取 /tmp/hello.txt
3. 寫入 /tmp/note1.txt 內容是 "note1"
4. 記低：我喜歡白色
5. 讀取 /tmp/note1.txt
6. 寫入 /tmp/note2.txt 內容是 "note2"
7. 搜尋記憶 拉麵
8. 記低：我周末要去野餐
9. 讀取 /tmp/note2.txt
10. 寫入 /tmp/note3.txt 內容是 "note3"
11. 搜尋記憶 白色
12. 記低：我有一隻貓叫 Mimi
13. 讀取 /tmp/note3.txt
14. 搜尋記憶 Mimi
15. 記低：我的最愛是薄荷綠茶
```

【預期】
- 正確辨識 15 個任務
- 所有檔案操作正確
- 最後合併報告
- 不超過 120 秒

### S2b — 混合類型大任務
```
我有很多野要做，幫我搞掃：
1. 讀取 /tmp/hello.txt 看下內容
2. 記低：我鍾意食拉麵
3. 用 TTS 讀出 "早晨好 the user" 的音頻
4. 搜尋記憶 拉麵看下有無記錄
5. 寫入 /tmp/summary.txt，內容包含今日日期同我的最愛食物
6. 讀取 /tmp/summary.txt 確認
7. 記低：我今日完成了大量工作
8. 搜尋記憶 工作
9. 寫入 /tmp/final.txt 內容是 "All done"
10. 讀取 /tmp/final.txt 確認
```

【預期】
- TTS 有實際發送音頻
- 檔案操作正確
- 記憶操作正確
- 不超過 120 秒

## 評分標準

| 結果 | 說明 |
|------|------|
| ✅ Pass | 所有回覆正常，焨有 crash |
| ⚠️ Partial | 有少許延遲或小錯誤，但未 crash |
| ❌ Fail | 有回覆異常、crash、或無限 loop |
| ⚡ Timeout | 某些消息超過 60 秒未回覆 |
