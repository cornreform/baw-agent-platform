📚 術語表

術語解釋
evolve.pyBAW 自我進化引擎核心。三層架構：L1 行為記錄、L2 模式分析、L3 自動優化
analyze()掃描最近 N 小時行為記錄，統計 tool call 成功率、錯誤模式、用戶修正
auto_optimize()根據 analyze 結果自動 patch SOUL.md 或 config.yaml。dry_run=True 時只報告不改檔案
selftest()端到端健康檢查：config、tool、memory、API key、網絡連通性
SchedulerCron-based 背景排程器，每 60 秒 poll 一次，支援 shell command 和 BAW skill
Watchdog持續監視系統健康狀態，異常時自動 alert（Phase 2 目標）
dry-run只模擬執行、輸出 diff，不實際修改檔案的安全模式
